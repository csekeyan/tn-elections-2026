ECI Data Pipeline (for May 4)

Known ECI URL pattern (from Delhi 2025)

https://results.eci.gov.in/ResultAcGen{Month}{Year}/ConstituencywiseS22{AC_NUMBER}.htm

For TN 2026, likely: ResultAcGenMay2026/ConstituencywiseS22{1-234}.htm

The transformer (data/transform_eci.py) parses ECI HTML pages and outputs the exact JSON format the dashboard expects. Tested and verified with Delhi 2025 data.

Architecture on counting day

Browser → Cloudflare Worker (/api/results)
              ↓ (cache: 30s TTL)
         ECI results.eci.gov.in
              ↓ (fetch 234 pages)
         Transform → JSON → Return to browser
