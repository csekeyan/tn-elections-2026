Architecture

Browser (thousands of users)
    |
    v
Cloudflare Pages (edge cache, 30s TTL)
    |
    |-- Static files: index.html, JS, CSS, images
    |-- /api/results → functions/api/results.js
    |       |
    |       v (cache MISS, once per 30s)
    |   ECI results.eci.gov.in
    |       |
    |       v
    |   Parse → JSON → Cache → Return
    |
    |-- /api/health → functions/api/health.js

ECI is hit at most once every 30 seconds. All users get the cached response. Zero load on ECI regardless of traffic.
